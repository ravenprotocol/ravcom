# ravcom
Raven Distribution Framework's common library to talk to databases

## Set environment variables
     # Redis

    RDF_REDIS_HOST="localhost"
    RDF_REDIS_PORT="6379"
    RDF_REDIS_DB="0"

    # MySQL
    RDF_MYSQL_HOST="localhost"
    RDF_MYSQL_PORT="3306"
    RDF_MYSQL_USER="root"
    RDF_MYSQL_PASSWORD="password"
    RDF_MYSQL_DATABASE="rdf"
